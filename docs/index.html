libgpio-manager

Library C/C++ for managment of GPIO pins on embedded Linux systems.

Overview

libgpio-manager provides a simple and efficient way to control GPIO pins on embedded Linux systems. It abstracts the complexities of GPIO handling and offers a user-friendly API for developers.

This library can be used in C/C++, and Python applications, making it versatile for various projects.

Features

  • Easy-to-use API for GPIO pin management
  • Actually supports only write and read operations
  • Supports multiple GPIO pins
  • Compatible still only Rock 5C board (but can be extended to other boards)

Example Usage

This example demonstrates how to make a simple blink in C.

C Example: Simple Blink

#include <stdio.h>
#include <unistd.h>
#include <gpio-manager/libgpio-manager.hpp>
int main(){
gpio_handle_t h = gpio_create();
if (!h)
{
printf("Failed to create GPIO handle\n");
return -1;
}
gpio_set_mode(h, "PIN_8", GPIO_DIR_OUTPUT);
for (int i = 0; i < 10; i++)
{
gpio_write(h, "PIN_8", HIGH);
usleep(500000); // 500 ms
gpio_write(h, "PIN_8", LOW);
usleep(500000); // 500 ms
}
gpio_destroy(h);
return 0;
}

This example demonstrates how to make a simple blink in Python.

Python Example: Simple Blink

import time
import ctypes
# Load the shared library
lib = ctypes.CDLL('libgpio-manager.so')
lib.gpio_create.restype = ctypes.c_void_p
# Create GPIO handle
h = lib.gpio_create()
if not h:
print("Failed to create GPIO handle")
exit(-1)
# Set pin mode to output
lib.gpio_set_mode(h, b"PIN_8", 1) # GPIO_DIR_OUTPUT
for i in range(10):
lib.gpio_write(h, b"PIN_8", 1) # HIGH
time.sleep(0.5) # 500 ms
lib.gpio_write(h, b"PIN_8", 0) # LOW
time.sleep(0.5) # 500 ms
# Destroy GPIO handle
lib.gpio_destroy(h)
